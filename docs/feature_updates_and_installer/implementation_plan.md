# YouTube Downloader 機能改善およびGUIインストーラー追加計画

## 概要
ダウンロード開始時の表示のクリア、画質選択の不具合修正、およびユーザーフレンドリーなGUIインストーラー/アンインストーラーを実装します。

## ユーザーレビューが必要な事項
> [!IMPORTANT]
> - GUIインストーラーはPythonの `tkinter` を使用して作成します。配布時は `PyInstaller` などで実行ファイル化することを想定しています。
> - 拡張機能のIDコピー誘導のため、インストーラー内に説明用のテキストとChrome拡張設定ページへのリンクボタンを設けます。

## 変更内容

### 1. 拡張機能のUI改善
#### [MODIFY] [panel.js](file:///c:/Users/haru/dev/YT/original/extension/panel.js)
- `detectVideo` 関数の開始時に、タイトルのテキストとサムネイルをクリアするように変更します。これにより、ページ遷移時に一瞬前の動画の情報が表示される「ゴースト」現象を防ぎます。
- ダウンロード開始後、または検出失敗時にUIの状態をリセットする処理を強化します。

### 2. ダウンロード品質の修正
#### [MODIFY] [host.py](file:///c:/Users/haru/dev/YT/original/host/host.py)
- MP4選択時のフォーマット指定を緩和します。現在は `ext=mp4` を強制しているため、1080pのMP4ストリームが存在しない場合に低画質が選ばれることがあります。
- 高画質なVP9などのビデオストリームも許可し、ffmpegを使用してMP4コンテナにマージするように修正します。

### 3. GUIインストーラー・アンインストーラー
#### [NEW] [installer_gui.py](file:///c:/Users/haru/dev/YT/original/host/installer_gui.py)
- `tkinter` を使用した **MSI風のウィザード形式（次へ、戻る、インストール等のステップ式）** のGUIインストーラー。
- 以下の機能を実装：
  - インストール先の選択。
  - ファイルのコピー（extension, host, yt-dlp, ffmpeg）。
  - レジストリへのNative Messaging Host登録。
  - 拡張機能IDの入力・設定。
  - ID取得方法の図解・説明。
#### [NEW] [uninstaller.py](file:///c:/Users/haru/dev/YT/original/host/uninstaller.py)
- レジストリの削除およびインストールしたファイルの削除を行うスクリプト。

## 検証計画

### 修正内容の確認
1. **キャッシュクリア**: 拡張機能を開き、別動画のページに移動した際に、前回のタイトルが残らず一旦クリアされることを確認。
2. **画質修正**: 1080pを選択してダウンロードし、成果物のファイルプロパティで解像度が1920x1080になっていることを確認。
3. **インストーラー**: `installer_gui.py` を実行し、GUIが表示され、インストール完了後にChromeで拡張機能が動作することを確認。
4. **アンインストーラー**: アンインストール実行後、レジストリとファイルが削除されることを確認。

### 手動検証
- ユーザーにGUIのデザインやIDコピーの誘導が分かりやすいかを確認いただく。
