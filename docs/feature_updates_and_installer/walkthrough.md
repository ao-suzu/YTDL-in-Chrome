# 修正完了・新機能追加のお知らせ

YouTube Downloaderの機能改善と、新しいインストーラーの実装を完了しました！

## 今回の変更点

### 1. ダウンロード開始時の表示・データクリア
- 動画検出を開始する際、表示だけでなく内部データ（タイトル等）も即座にクリアするようにしました。
- データの取得が完了してタイトルが正しく表示されるまで、**ダウンロードボタンを一時的に無効化**するようにしました。これにより、タイトルがURLのままダウンロードされてしまう問題を防ぎます。

### 2. 画質選択のさらなる改善 (1080p合体エラーの解決)
- YouTubeの1080p動画に多い「MP4コンテナに非対応な音声形式（Opus等）」をダウンロードした場合でも、**合体時に自動でAAC音声へ変換する**ようにロジックを強化しました。これにより、1080pを選んだ時に「合体エラーで0KBになる」問題や「音が鳴らない」問題が完全に解消されました。
- H.264/AACといった、より汎用的な形式のストリームを優先的に取得するように `format_sort` を最適化しました。

### 3. MSI風GUIインストーラー
一般的なWindowsソフトのようなウィザード形式のインストーラーを実装しました。
- インストール先の指定
- 拡張機能IDの入力・誘導
- 自動的なレジストリ登録
- 設定ファイルの自動生成（Pythonパスの解決など）

### 4. アンインストーラー
レジストリ設定などを削除するアンインストーラーも同梱しています。

## 使い方
1. `host/installer_gui.py` を実行してください。（配布時はexe化を推奨）
2. ウィザードに従って進め、拡張機能のIDを入力してください。
3. インストール完了後、ブラウザで拡張機能を読み込み直せば準備完了です！
